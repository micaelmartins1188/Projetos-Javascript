Fórmula de Bhaskara

A fórmula de Bhaskara é uma fórmula matemática usada para encontrar as raízes de equações do segundo grau, do tipo ax2 + bx + x = 0, onde a, b e c são coeficientes reais.

A fórmula é dada por:

x = - b ± b 2 - 4 . a . c 2 . a

Etapa 1: Achar a discriminante

Δ = b 2 - 4 . a . c

Δ > 0 A equação terá duas raízes reais e distintas

Δ = 0 A equação apresentará uma raiz real

Δ < 0 A equação não possui raízes reais

Etapa 2: Altere a fórmula de Bhaskara

x = - b ± Δ 2 . a

Etapa 3: Calcule as raízes da equação

x 1 = - b + Δ 2 . a x 2 = - b - Δ 2 . a

Resultado

Δ = b 2 - 4 x a x c
Δ = b + 4
Δ = 4
x 1 = - b + Δ 2 x a
x 1 = bΔ + oi 2a
x 1 = bΔ 2a
x 1 = bΔ2a
x 2 = - b - Δ 2 x a
x 2 = bΔ - oi 2a
x 2 = bΔ 2a
x 2 = bΔ2a
